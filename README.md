 System & Network Dashboard

Présentation
Développement d'une interface web de monitoring permettant de visualiser l'état de santé d'un système. Ce projet combine programmation backend (Python) et intégration frontend (HTML/CSS).

Stack Technique
- Backend : Python Flask (API REST simple).
- Frontend: HTML5 / CSS3 (Design "Dark Mode" orienté NOC/Cyber).
- Logiciel : Utilisation de JSON pour l'échange de données.

 Fonctionnalités
- Affichage des statistiques CPU/RAM en temps réel.
- Monitoring visuel des connexions réseau actives.
- Interface responsive pour une utilisation sur console de supervision.

Objectif Pédagogique
Démontrer la capacité à créer des outils d'administration système personnalisés pour faciliter la prise de décision rapide en environnement réseau.
